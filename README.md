# Daily Todo List with Routines 📝

Vanilla JS와 Firebase를 활용하여 만든 **실시간 동기화 투두리스트**입니다.
PC와 모바일 어디서든 접속하여 할 일을 관리하세오.

![Preview](https://via.placeholder.com/800x400?text=Daily+Todo+Preview)

## ✨ 주요 기능

*   **📅 날짜별 관리**: 매일매일 새로운 페이지에서 할 일을 기록합니다.
*   **🔄 매일 루틴(Routine)**: 한번 등록하면 매일 자동으로 리스트에 뜨는 '루틴' 기능.
*   **☁️ 실시간 동기화**: Firebase 연동으로 PC에서 쓰고 폰에서 바로 확인.
*   **🔗 간편한 공유**: 링크만 보내면 친구, 가족, 동료와 리스트 공유 가능.
*   **📱 모바일 최적화**: 폰에서도 앱처럼 깔끔하게 보이는 반응형 디자인.
*   **🌙 다크 모드**: 눈이 편안한 딥 다크 + 네온 글래스모피즘 디자인.

## 🛠 사용 기술

*   **Frontend**: HTML5, CSS3, Vanilla JavaScript
*   **Backend**: Google Firebase (Firestore Database)

## 🚀 사용 방법

1.  상단 **'루틴 관리'**에서 매일 할 일을 등록하세요. (예: 영양제, 운동)
2.  **'공유하기'** 버튼을 눌러 링크를 복사하세요.
3.  링크를 카카오톡 '나와의 채팅' 등에 저장해두고 폰으로 접속하세요.
